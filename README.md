# Traveller Tools
A set of digital tools for the tabletop roleplaying game Traveller (Mongoose 2e).

## Project setup

Requires npm 6.14.8+
Requires node v14.15.0+

```
npm install
```

### Available Scripts
Runs the app in the development mode.
```
npm start
```
Launches the test runner in the interactive watch mode.
```
npm test
```
